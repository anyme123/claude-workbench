Claude Workbench v4.1.3 - Portable Version
========================================

Usage:
1. Double click claude-workbench.exe to start

Configure Acemcp Project Context Search (Optional):
1. Create config file: %USERPROFILE%\.acemcp\settings.toml
2. Add the following content:
   BASE_URL = "https://your-api-endpoint.com"
   TOKEN = "your-api-token"

Directory Structure:
  claude-workbench.exe          - Main application (11MB)
  binaries\
    acemcp-sidecar-*.exe        - Semantic search engine (35MB)

Documentation: https://github.com/anyme123/claude-workbench

NO EXTRA INSTALLATION REQUIRED - READY TO USE!
