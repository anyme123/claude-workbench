# 任务完成检查清单

## 代码修改后必须执行的步骤

### 1. TypeScript类型检查（必须）
```bash
npx tsc --noEmit
```
**说明**: 严格的TypeScript检查确保代码类型安全，防止运行时错误

### 2. Rust代码检查
```bash
cd src-tauri
cargo check
cargo clippy
```
**说明**: 检查Rust编译错误和最佳实践建议

### 3. 生产构建测试（关键）
```bash
bun run tauri build
```
**说明**: 开发模式和生产模式行为可能不同，必须用生产构建测试

### 4. 功能验证
- 测试修改的具体功能
