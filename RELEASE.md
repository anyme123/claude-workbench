## 📦 安装方式

### 预构建版本

从 [GitHub Releases](https://github.com/anyme123/claude-workbench/releases/tag/v4.0.0) 下载：

**Windows**:
- MSI 安装包
- NSIS 安装包

**macOS**:
- DMG 安装包
- APP 应用包

### 源码构建

```bash
git clone https://github.com/anyme123/claude-workbench.git
cd claude-workbench
git checkout v4.0.0

# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建应用
npm run tauri build
```
